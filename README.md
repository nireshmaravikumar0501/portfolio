# Personal Portfolio Website

A responsive personal portfolio website built using React and Vite to showcase my projects, technical skills, experience, and achievements.

## Features
- Responsive modern UI
- About Me section
- Skills and achievements
- Experience timeline
- Project showcase
- Contact section
- GitHub and LinkedIn integration

## Tech Stack
- React.js
- Vite
- JavaScript
- CSS / Tailwind (update if applicable)
